# Food-and-Travel---Zaayaka
Zaayaka 🌶️🍽️

Zaayaka is a Food & Travel website that brings together the rich diversity of India’s cuisine and culture into one digital experience. The project is designed to let users explore different regions of India through their signature dishes, local flavors, and popular travel destinations.

Built as part of my Web Development journey (Task 1), this project focuses on creating a visually appealing, responsive, and user-friendly interface while strengthening core frontend skills.

🔍 Features:
- Explore regional food and travel destinations across India
- Clean and responsive UI for smooth navigation
- Interactive elements using JavaScript
- Organized layout with structured sections

💻 Tech Stack:
- HTML
- CSS
- JavaScript

🎯 Objective:
To combine creativity with functionality by building a platform that showcases India’s food and travel culture while improving frontend development fundamentals.

This project marks the beginning of my journey in web development, with many more features and improvements planned ahead 🚀
